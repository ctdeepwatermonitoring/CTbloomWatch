Connecticut bloomWatch

bloomWatch is an U.S. EPA crowdsourcing application that allows the public to report blooms of cyanobacteria. This map displays the blooms reported in Connecticut waterbodies. Please note that the blooms reported are not verified. For information on State beach closures go to CT DEEP website. For town beach closures contact your local health district.

Move sliders to return bloom observations for selected date range.
