Connecticut bloomWatch

bloomWatch is an U.S. EPA crowdsourcing application that allows the public to report blooms of cyanobacteria. This map displays the blooms reported in Connecticut waterbodies. Please note that not all blooms reported are verified. For information on State beach closures please see the State Swimming Report. For information on town beach closures contact your local health district.

Move sliders to return bloom observations for selected date range.
